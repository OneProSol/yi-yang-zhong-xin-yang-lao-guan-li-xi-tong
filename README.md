# 颐养中心养老管理系统

一个基于Vue3 + SpringBoot + MyBatisPlus + MySQL的颐养中心养老管理系统。

### 一、相关技术
- 后端：`jdk`、`SpringBoot 2.x`、`MyBatisPlus 3.x`、`JWT`。
- 前端：`node`、`Vue 3`、`Vue Router`、`Element Plus`、`Axios`。
- 数据库：`mysql`。

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk17 或 jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm` 。
- 数据库：`mysql5.7以上`。

### 四、功能描述及功能图展示

系统分为：护理、管理员。

护理功能：1.登录、2.首页、3.退住登记、4.外出登记、5.护理记录、6.日常护理

管理员功能：1.登录、2.首页、3.入住登记、4.退住登记、5.外出登记、6.床位示意图、7.床位管理、8.护理项目、9.护理级别、10.客户护理设置、11.护理记录、12.设置服务对象、13.服务关注、14.用户管理
